Bot De Mort

Plus que avant le combat

FAITES VOS JEUX !

Projet réalisé dans le cadre du Tremplin numérique n°6.
Il s'agit de fabriquer un robot qui combattra dans une arêne.
Le premier robot qui fait tomber son canard (voir si dessous) perd le combat.

Étapes

Étape 1 : Schéma

Pour commencer nous avons débattu sur la forme du robot et fait des schémas.

Étape 2 : Arduino

J’ai branché les 3 potentiomètres avec des fils Dupont sur la breadboard et la carte Arduino.

J’ai connecté deux fils aux rails + et – de la breadboard.

Étape 2 : Arduino

Les 3 servos sont branchés sur les broches 2, 8, 13 et reliés aux potentiomètres.

J’ai connecté la carte Arduino en USB à l’ordinateur, puis utilisé l’IDE Arduino pour y téléverser le code.

Étape 3 : Set up Git

On a essayé de créer un repository sur Git pour pouvoir travailler à deux sur le site.

Étape 4 : Site Web

On a créé la base du site pour pouvoir y mettre la documentation au fur et à mesure.

Étape 5 : Prototype

On a créé un prototype en carton pour voir à quoi il ressemblera et quelle taille il fera.

Étape 6 : Logo Robot

On a demandé à une IA de faire un logo en lui donnant notre thème et couleur.

Étape 7 : Découpe laser

On a découper en bois les pièces nécessaire au montage.

On a assemblé toutes les pièces en bois à l'aide d'un maillet.

Étape 9 : Peindre le robot

Les contraintes :

  • Le gabarit du robot ne doit pas dépasser les 20*20*20 cm
  • Un canard doit tenir en équilibre sur son point le plus haut
  • Le thème imposé est Harry Potter en noir et blanc
  • Le robot doit contenir au moins 3 potentiomètres et trois servomoteurs

Code

Afficher le code ! 

          #include 

          // Define les broches pour les servo-moteurs
          const int servoPin1 = 8;
          const int servoPin2roue = 13;
          const int servoPin3 = 2;

          // Define les broches pour les potentiomètres
          const int potPin1 = A0;
          const int potPin2 = A1;
          const int potPin3 = A2;

          // Crée des objets Servo pour les servo-moteurs
          Servo servo1;
          Servo servo2;
          Servo servo3;

          void setup() {
            // Initialise les servo-moteurs
            servo1.attach(servoPin1);
            servo2.attach(servoPin2roue);
            servo3.attach(servoPin3);
          }

          void loop() {
            // Lit les valeurs des potentiomètres
            int angle1 = map(analogRead(potPin1), 0, 1023, 0, 180);
            int angle2 = map(analogRead(potPin2), 0, 1023, 0, 180);
            int angle3 = map(analogRead(potPin3), 0, 1023, 0, 180);

            // Déplace les servo-moteurs aux positions spécifiées par les potentiomètres
            servo1.write(angle1);
            delay(15); // Ajout d'un délai de 15 millisecondes
            servo2.write(angle2);
            delay(50); // Ajout d'un délai de 15 millisecondes
            servo3.write(angle3);
            delay(15); // Ajout d'un délai de 15 millisecondes
          }
